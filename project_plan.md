# План реализации проекта: GETMANENKO DESSERT

Этот документ фиксирует весь план разработки, который был успешно реализован на сайте-портфолио кондитерской студии.

---

## 📋 Выполненные этапы проекта

### 1. Подготовка и оптимизация медиа-ресурсов
*   **Логотип**: Интегрирован файл `logo_desert.png` в шапку и подвал сайта в качестве официального логотипа.
*   **Сжатие графики**: Все 60 изображений тортов (из папки `tort/`) и начинок (из папки `fillings/`) были оптимизированы и сжаты с помощью Python-библиотеки Pillow. Это сократило вес сайта на **35%** (с 6.7 МБ до 4.4 МБ) и обеспечило его мгновенную загрузку даже при медленном мобильном интернете.
*   **Исключение мусора**: Настроены файлы `.gitignore` и `.surgeignore`, чтобы при публикации и отправке в Git не загружались временные файлы, кэши и скриншоты.

### 2. Новый визуальный функционал и эстетика
*   **Шрифты**: Интегрирован красивый контрастный шрифт **Inter** с полноценной поддержкой кириллицы.
*   **Lux Dark Mode**: Премиальная темная тема. Состояние переключателя (Солнце/Луна) сохраняется в `localStorage` браузера, а иконка смены тем плавно вращается при клике.
*   **3D Tilt-эффект**: Карточки начинок и элементы галереи плавно наклоняются вслед за курсором мыши, имитируя трехмерный объем, с динамическим бликом (`.tilt-shine`), рассчитывающим направление света.
*   **Instagram-галерея**: Старый горизонтальный слайдер заменен на адаптивную сетку с вкладками-фильтрами:
    *   *Свадебные торты*
    *   *Детские торты*
    *   *Праздничные / Тематические торты*
*   **Лайтбокс (Просмотрщик)**: Полноэкранный просмотр фото с подписями категорий, возможностью перелистывания жестами (на телефонах) и кнопками клавиатуры (на ПК).

### 3. Оптимизация ассортимента
*   **Начинки**: Список начинок сокращен с 29 до ровно 25 самых востребованных позиций согласно ТЗ. Добавлен удобный живой поиск по ингредиентам.
*   **Новый раздел «Десерты»**: Презентован постоянный ассортимент. Главная позиция — фирменный *Сырный пирог с грушей* с качественными ИИ-изображениями (целиком и в разрезе) и возможностью быстрого заказа.

### 4. Интерактивное видео
*   В секции Hero добавлена кнопка воспроизведения промо-видео с эффектом пульсации. При нажатии открывается модальное окно с плеером, а при закрытии — видео автоматически останавливается (сбрасывается iframe), чтобы не продолжался звук.

### 5. Облачная CMS (Firebase) и Админка
*   **Пароль по умолчанию**: `admin`
*   Вход в админку доступен по скрытой кнопке (иконка замочка в шапке или ссылка в подвале).
*   В режиме администратора на сайте появляются кнопки «Добавить» и «Удалить» для начинок, десертов и галереи.
*   **Загрузка файлов**: Реализована возможность загружать фото файлов прямо с телефона/ПК.
*   **Режимы CMS**:
    *   *Автономный*: Сохранение данных в `localStorage` ( Base64 для картинок).
    *   *Облачный*: Полная синхронизация в реальном времени с базами данных **Cloud Firestore** и хранилищем картинок **Firebase Storage** (инструкция по подключению ключей находится в `README.md`).
